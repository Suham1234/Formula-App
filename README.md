# HSC Formula Flashcard App

A modern dark-themed flashcard application for Bangladesh HSC students to practice and master Math, Physics, and Chemistry formulas.

## Features

- **Flip Cards Mode**: Click to reveal formula and explanation
- **Type Answer Mode**: Test your knowledge by typing formulas
- **Subject & Topic Organization**: Browse by subject and specific topics
- **Progress Tracking**: Mark formulas as "Learned" or "Needs Practice"
- **Search**: Quickly find specific formulas
- **Flexible Matching**: Type answers with flexible formatting (spaces, symbols don't need to be exact)
- **Dark Theme**: Modern, easy on the eyes design

## How to Run

1. Open `index.html` in any modern web browser
2. Or use a local server (recommended for ES modules):

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then visit `http://localhost:8000`
